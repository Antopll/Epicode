# 📊 Epicode – Data Analyst Course

Repository con gli esercizi e i progetti svolti durante il percorso 
**Data Analyst** di Epicode (giugno 2026 – gennaio 2027)

## 🎯 Obiettivo

Questo percorso rappresenta un tassello del mio percorso di crescita professionale: 
partendo da un ruolo operativo in Amazon (sortation associate), sto 
costruendo competenze tecniche in analisi dati con l'obiettivo di transitare verso 
un ruolo di **Data Analyst**, con un interesse specifico per l'applicazione di 
queste competenze all'ambito **sportivo/calcistico** (scouting e performance analysis).
